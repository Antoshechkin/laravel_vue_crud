## О проекте

Технологии:
- Backend - PHP 8, Laravel 8
- FrondEnd - Vue 2
- DataBase - MySQL

Проект выполнен как демонстрация технологий. Аутентификация реализована штатными средствами Laravel Sanctum. В проекте реализована одна сущность People, для которой реализован CRUD.

Расположения:
- app/Models - ORM
- app/Http/Controllers - Контроллеры
- database/migrations - Миграции
- resources/js/ - Vue
- resources/views/ - Шаблоны Blade
- routes/api.php - Api Routes
- routes/web.php - Web Routes
